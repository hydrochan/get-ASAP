    1. **deploy.sh (per D-07, D-09):**
       ```bash
       #!/usr/bin/env bash
       set -euo pipefail
       ```
       스크립트 내용:
       - echo로 각 단계 안내 출력
       - Python 3.11+ 버전 체크: `python3 --version`
       - venv 생성: `python3 -m venv .venv` (이미 있으면 스킵)
       - venv 활성화: `source .venv/bin/activate`
       - pip install: `.venv/bin/pip install -r requirements.txt`
       - logs 디렉토리 생성: `mkdir -p logs`
       - .env 파일 존재 체크: 없으면 `.env.example`을 `.env`로 복사하라고 안내
       - token.json 존재 체크: 없으면 SCP로 전송하라고 안내 (per D-08)
       - credentials.json 존재 체크: 없으면 SCP로 전송하라고 안내
       - 테스트 실행 안내: `.venv/bin/python -m pytest tests/ -v`
       - 드라이런 안내: `.venv/bin/python main.py --dry-run --verbose`
       - crontab 설정 안내 출력 (per D-04, D-05):
         ```
         echo "=== crontab 설정 ==="
         echo "crontab -e 로 편집기 열고 아래 줄 추가:"
         SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
         echo "0 */6 * * * cd ${SCRIPT_DIR} && ${SCRIPT_DIR}/.venv/bin/python main.py >> logs/cron.log 2>&1"
         ```
       - chmod +x deploy.sh 안내는 불필요 (스크립트 자체에 실행 권한)

    2. **.env.example:**
       config.py에서 사용하는 모든 환경변수를 빈 값 + 주석으로 나열:
       ```
       # Gmail OAuth 설정
       GMAIL_CREDENTIALS_PATH=credentials.json
       GMAIL_TOKEN_PATH=token.json

       # Notion 설정
       NOTION_TOKEN=
       NOTION_DATABASE_ID=
       NOTION_PARENT_PAGE_ID=

       # 실행 설정
       CHECK_INTERVAL_HOURS=6
       ```

    3. **.gitignore 업데이트:**
       기존 .gitignore 읽고, 없는 항목만 추가:
       ```
       # Secrets
       .env
       token.json
       credentials.json

       # Python
       __pycache__/
       *.pyc
       .venv/

       # Logs
       logs/

       # State
       state.json

       # OS
       .DS_Store
       Thumbs.db
       ```

    deploy.sh에 실행 권한 설정: `chmod +x deploy.sh` (bash tool로 실행)
