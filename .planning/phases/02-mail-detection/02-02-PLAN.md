       - state에 historyId 없으면: messages.list(q=query, maxResults=1) 호출 -> 첫 메시지 historyId를 state에 저장 -> 메시지 ID 목록 반환

       messages.modify(addLabelIds=[label_id], removeLabelIds=["UNREAD"]).

    7. `extract_body(payload: dict) -> str`:
       Gmail 메시지 payload에서 HTML/text 본문 추출. base64.urlsafe_b64decode(data + "==") 사용.
       multipart 재귀 처리. text/html 우선, 없으면 text/plain.

    8. `infer_journal(sender: str, subject: str, publishers: dict) -> str` (per D-06):
       publishers dict에서 sender 매칭으로 출판사 찾기 -> journals 리스트의 각 이름이 subject에 포함되는지 검사 -> 매칭되면 해당 저널명 반환, 아니면 출판사 name 반환.
