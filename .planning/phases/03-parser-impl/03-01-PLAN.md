구현할 함수:
1. `save_fixture(publisher_key: str, sender: str, max_count: int = 2) -> list[str]`:
   - `auth.get_gmail_service()`로 Gmail 서비스 객체를 얻는다
   - `service.users().messages().list(userId="me", q=f"from:{sender}", maxResults=max_count).execute()`로 메일 검색
   - 각 메일에 대해 `service.users().messages().get(userId="me", id=msg["id"], format="full").execute()`로 전체 payload 가져옴
   - `gmail_client.extract_body(full_msg["payload"])`로 HTML 본문 추출
   - 메일 헤더에서 실제 From 주소를 추출하여 출력: `full_msg["payload"]["headers"]`에서 `name == "From"` 항목의 `value`
   - `tests/fixtures/{publisher_key}_{idx:02d}.html`로 저장 (UTF-8)
   - 저장 경로 목록 반환

2. `verify_senders(service, publishers: dict) -> dict`:
   - 각 출판사 sender로 `messages.list(q=f"from:{sender}", maxResults=1)` 호출
   - 실제 메일의 From 헤더를 추출하여 publishers.json의 sender와 비교
   - 불일치 항목을 `{publisher_key: {"expected": sender, "actual": real_sender}}` 형태로 반환
   - 일치하면 빈 dict 반환

3. `main()`:
   - `publishers.json`을 `json.load`로 읽기
   - `verify_senders()`로 sender 검증 -> 불일치 시 경고 출력 및 publishers.json 자동 수정
   - 각 출판사에 대해 `save_fixture(key, sender, max_count=2)` 호출
   - `os.makedirs("tests/fixtures", exist_ok=True)` 사전 호출
   - 결과 요약 출력: 출판사별 저장 파일 수, 총 바이트 수

`if __name__ == "__main__": main()` 블록 포함.

- 수정 시 원본 값과 새 값을 `print(f"[수정] {key}: {old} -> {new}")` 형태로 출력
