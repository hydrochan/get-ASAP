<critical_api_notes>
## notion-client 3.0.0 Breaking Changes (MUST READ)

1. `client.databases.query()` 가 존재하지 않음 — AttributeError 발생
2. 대신 `client.data_sources.query(data_source_id, filter=...)` 사용
3. data_source_id 획득: `client.databases.retrieve(db_id)["data_sources"][0]["id"]`
4. DB 생성 시 스키마는 `initial_data_source={"properties": {...}}` 아래에 위치
5. `pages.create`는 변경 없음 — `parent={"database_id": db_id}` 그대로 사용
6. Title 속성 필터: `"title": {"contains": ...}` (rich_text 아님)
</critical_api_notes>
